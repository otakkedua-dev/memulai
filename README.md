# 📊 Python Sorting Algorithms Collection

Repositori ini berisi kumpulan implementasi algoritma pengurutan data (*sorting*) menggunakan bahasa pemrograman Python. Proyek ini bertujuan untuk menjadi bahan pembelajaran dan referensi mengenai cara kerja berbagai algoritma *sorting* dari yang paling dasar hingga yang paling efisien.

## 🚀 Pembaruan Terbaru

- **[BARU]** Baru saja ditambahkan **1 algoritma sort baru** ke dalam koleksi ini, yaitu **[Bubble Sort]**. Algoritma ini sudah dilengkapi dengan contoh implementasi kode dan penjelasan singkat di dalam file sumbernya.
